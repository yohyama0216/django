## プロジェクトの作成
### django自体のプロジェクトの作成
django-admin startproject myproject

### アプリケーションの作成
python manage.py startapp helloworld
機能ごとに作るらしい？
 Djangoの「アプリケーション」という概念は、ウェブアプリケーションの開発をモジュラーで再利用可能なコンポーネントに分割することを促し、大規模なプロジェクトの管理を容易にする設計哲学の一部です。
